CSS position属性解析

Posted on 2017-07-12

在学习CSS时被position属性搞晕了，因为这个属性很重要，所以专门研究了一下。

想要了解position属性，首先要明白一个概念–“流”。

流（Flow）

流实际上就是浏览器在页面上摆放HTML元素所用的方法。浏览器从HTML文件最上面开始，从上到下沿着元素流逐个显示 所遇到的各个元素。先考虑块元素，它会在每个块元素之间加上一个换行。所以首先会显示文档中的第一个元素，然后是换行。然后是第二个元素，接下来又是一个换行。如此继续，从文件最上面一直到文件末尾逐个显示。而内联元素在水平方向上会相互挨着，总体上会从左上方流向右下方。这就是流。 了解了流再来看一下position属性。

position属性

position属性规定了元素的定位类型。可以取得的值有absolute、relative、static、fixed、inherit。默认的值是static。这个属性是不可继承的。该属性各个值的用法如下

值 描述
absolute 生成绝对定位的元素，相对于 static 定位以外的第一个父元素进行定位。元素的位置通过 “left”, “top”, “right” 以及 “bottom” 属性进行规定。
relative 生成相对定位的元素，相对于其正常位置进行定位。因此，”left:20” 会向元素的 LEFT 位置添加 20 像素。
static 默认值。没有定位，元素出现在正常的流中（忽略 top, bottom, left, right 或者 z-index 声明）。
fixed 生成绝对定位的元素，相对于浏览器窗口进行定位。元素的位置通过 “left”, “top”, “right” 以及 “bottom” 属性进行规定。
inherit 规定应该从父元素继承 position 属性的值。

static

static是position属性的默认值，此时元素会按照默认的方式，即“流”的方式排列在页面上，每一个元素都会占用一定的空间，此时top, bottom, left, right属性都失去作用。

relative

relative是生成相对定位的元素，这个相对是相对于其正常的位置，即默认情况下出现在流中的位置。设置了top、left等属性后，元素就会相对其正常位置进行定位，定位的规则是，为哪个方向指定多少像素，就在哪个方向上添加多少像素。例如，”left:20” 会向元素的 LEFT 位置添加 20 像素，意即元素相对于正常位置向右平移了20像素。并且，在元素定位之后，其在新位置上不占据空间，而在原本位置上仍然占据着原有的空间，其他元素按“流”来排列布局时，依然不能忽略它，就好像它还存在一样。

absolute

absolute是生成绝对定位的元素，这个绝对的意思是，绝对定位的元素会被从”流“中取出，元素定位后，其在新位置上同样不占据空间，在原本位置上也不占据空间，也就是说此时其并不存在于流中，对其他元素的排列没有任何影响。另外，absolute的定位方式是相对于 static 定位以外的第一个父元素进行定位。如果父类元素的定位方式为static，就向上找祖先元素，直到找到非static定位的元素或是html元素，然后相对这个祖先元素进行定位。所以在使用absolute的时候会和relative配合使用，如果需要使一个元素相对于其祖先元素进行定位，需要先指定其祖先元素为relative定位，然后对其进行绝对定位。

图解

图中大框都是宽高200px，小框都是宽高100px。源代码在这里

红色虚线框———relative定位

图中红色虚线框使用了relative定位，并且将left设置为200px,top设置为100px。可以看到其相对原始位置（红色实线框位置）向右移动了200px，并向下移动了100px。并且可以看到绿色实线框在流中依然察觉到了红色虚线框的存在，排在了其原始位置的正下方（而不是当前红色虚线框的下方。说明红色虚线框在流中依然占据着原来的空间，但在当前位置上并不占据空间。

蓝色实线框和橙色实线框———absolute定位

图中，橙色实线框是绿色实线框的子元素，设置为absolute定位，top值设置为20px.根据absolute定位的原则，由于其祖先元素不存在非static定位，所以其相对于html元素进行定位。定位位置为距离html元素顶部100px的位置。 蓝色实线框是红色虚线框的子元素，设置为absoulute定位，top值设置为100px。由于其父元素（红色虚线框是）relative定位，所以相对红色虚线框进行定位，定位位置为距离红色虚线框上边框100px位置。
