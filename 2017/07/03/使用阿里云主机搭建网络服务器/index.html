安装FTP软件

通过FTP软件将网站文件上传到云服务器。这里选择FileZilla这款免费FTP软件。 打开FileZilla，输入公IP、用户名、密码然后链接到云服务器，可以查看云服务器的根目录。 默认情况下，Apache服务器的网站目录在/var/www,所以将网站文件上传到该目录下，在浏览器地址栏中输入云服务器的公网IP，就可以看到自己的网站了。
