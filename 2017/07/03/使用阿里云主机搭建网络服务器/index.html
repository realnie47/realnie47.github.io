使用阿里云服务器搭建网络服务器

Posted on 2017-07-03 |

想要建一个网站，大概知道了建站的步骤，无非是搭建网络服务器、把网站的文件放在网络服务器上，买一个域名，域名解析到自己服务器的ip…可是突然想到一个问题，为什么要把网站放在虚拟主机（比如阿里云主机）上，放在自己的PC上不好吗，省钱还便于操作。偶然在知乎上看到了各位大神的回答。答案是可以的，只要你想，用手机都可以做服务器。当然这是有缺点的的，简单的来讲就是三个字：不稳定。小白想要建站，还是老老实实买云服务器吧。
大概记录下使用阿里云服务器搭建网络服务器的步骤。

购买云服务器

在阿里云购买一个云服务器，选择服务器的配置，主要是最后付款。付款成功后就拥有了一个云服务器。因为是实验性质的，我选择了最基础的配置，没什么选择的余地，半年只需9元。只有一个需要选的就是云服务器的操作系统。我选择了linux（centOS）。

安装FTP软件

通过FTP软件将网站文件上传到云服务器。这里选择FileZilla这款免费FTP软件。
打开FileZilla，输入公IP、用户名、密码然后链接到云服务器，可以查看云服务器的根目录。
默认情况下，Apache服务器的网站目录在/var/www,所以将网站文件上传到该目录下，在浏览器地址栏中输入云服务器的公网IP，就可以看到自己的网站了。
