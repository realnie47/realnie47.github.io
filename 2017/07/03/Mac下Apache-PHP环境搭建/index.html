Posted on 2017-07-03 |

在学习前端时，需要搭建一个网页服务器环境，Mac自带了Apache服务器，只需开启并做简单的配置就可以使用了，具体步骤如下。

  • Apache版本：2.4.25
  • PHP版本：5.6.30
