  2. 安装一个静态网站框架。这里选择Hexo。关于什么是Hexo及其如何安装可以参考文档 | Hexo。

  5. 创建文章。可以使用如下命令创建文章：

  6. 编辑文章。使用文本编辑器编辑文章。

  7. 开启Hexo服务。必须开启Hexo服务才能在本地实时查看生成的网页。首先安装hexo-server：
    $ npm install hexo-server --save
    输入如下命令启动服务器：
    $ hexo server或$ hexo s
  8. 预览本地效果。 在浏览器地址栏输入http://localhost:4000，查看实时生成的网站效果。
